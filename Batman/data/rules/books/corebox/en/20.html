HERO MOVEMENT
The movement action consists of one or several
moves made using move points. The move point
pool corresponds to the number of move points the
miniature can spend.
Declare a Movement
The hero declares a movement.
Create the Move Point Pool
The move point pool equals zero plus eventual
bonuses conferred by skills, traits and special rules.
Add Move Point Bonus for the First
Movement
If this movement action is the ﬁrst of the hero’s turn
for this miniature, they add their move point bonus
to their move point pool.
Choose Move Type
The area in which the miniature is located is the
start area. The hero chooses the area to which the
miniature will move. This area is the destination
area. A miniature cannot enter an area if by doing
so they would over-occupy it (
see page 43
). The des-
tination area determines the type of move required:
