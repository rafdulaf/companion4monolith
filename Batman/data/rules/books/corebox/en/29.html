THE TILES
There are two categories of tiles:
-
character tiles
-
event tiles.
The character tiles
Each tile represents one or more characters that can be
controlled. Usually controlled by the villain, using their
command post, however some heroes can control charac-
ters associated with a tile using their elite command skill
or henchman command skill (
see page 48
).
