7 Nouveaux types de cartes Équipement La carte Équipement Corde introduit une nouvelle mécanique. Un Héros possédant une carte Corde peut réaliser une action de Manipulation simple pour la défausser et placer le pion Corde sur le plateau comme défini par le scénario. Un Héros dans la même zone que le pion Corde peut réaliser une Manipulation simple pour retirer ce pion du plateau et placer la carte Corde dans son inventaire. Le Sceptre de Jade utilise l’action de Manipulation pour permettre à son porteur de lancer un rayon dévastateur. Cet artefact cause des dégâts inévitables à une seule cible. Pour les Héros, ceci demande la dépense de gemmes sur la case Manipulation. Les unités de l’Overlord ne disposant pas de case Manipulation, l’Overlord dépense simplement ses gemmes de la Zone d’Énergie disponible vers la Zone de Fatigue. Les Shuriken sont des armes légères à courte portée. Elles suivent toutes les règles de base des armes à distance (ligne de vue, phénomène de gêne...), mais elles ne peuvent être utilisées que pour cibler un ennemi dans la même zone que l’attaquant ou dans une zone adjacente. Cela comprend les zones d’élévations différentes, ce qui peut entraîner un bonus de surplomb. Le Fouet est une arme qui peut aussi être utilisée comme une carte Corde lorsque le scénario indique qu’un pion Corde peut être placé. Nouveaux types de Tuiles Nous avons créé de nouvelles tuiles Unités afin de tirer davantage parti des figurines déjà existantes. Certaines, comme le Capitaine stygien ou les Guerriers kushites, sont à utiliser dans les nouveaux scénarios en remplacement direct des tuiles existantes. D’autres, comme les tuiles Sans-âmes ou les Tlatizlans permettent l’utilisation de plusieurs types de figurines. Par exemple, les Sans-âmes peuvent être représentés par des figurines de Mommies, de Squelettes, ou un mélange des deux. Quelque soit le type de figurines choisi, elles auront toutes le profil indiqué par la tuile. Nouveau type de Carte Sort Linceul Spectral est un nouveau type de sort défensif qui permet à son lanceur de bloquer jusqu’à 3 dégâts, quelle qu’en soit la cause : attaque, chute, dégât inévitable... Si des pions sont présents sur la carte Sort, leur utilisation est obligatoire en cas de dégât subi. Ce sort peut être lancé à nouveau avant que tous les pions aient été utilisés pour remonter à 3 pions présents sur la carte. Cartes Scénario Un des scénarios de ce livret, La Porte de l’Aigle (p. 30), introduit un nouvel élément de rejouabilité pour le Mode Aventure sous la forme de cartes. Ni l’Overlord, ni les Héros ne sauront ce que dissimulent les portes closes. Les joueurs pourront placer sur le plateau les pions numérotés ou les cartes elles-mêmes qui ne seront révélées que lorsqu’un Héros ouvrira la porte correspondante. Ces cartes ont deux effets : l’apparition immédiate sur le plateau de nouveaux ennemis et un effet permanent à l’avantage de l’Overlord (ou au désavantage des Héros). Les cartes ne sont pas toutes utilisées dans un scénario, ce qui permet de ne jamais être certain de quels ennemis seront présents, ni de l’endroit où ils apparaîtront.
